# GTNH GregTech ru_RU

Ресурспак с русской локализацией материалов GregTech и аддонов для модпака [GregTech: New Horizons](https://github.com/GTNewHorizons).

## Покрытие

| Аддон | Описание |
|-------|----------|
| GregTech | Базовые материалы, руды, провода, трубы и предметы |
| BartWorks | Материалы аддона BartWorks |
| GT++ (miscutils) | Материалы аддона GT++ |

## Установка

1. Скачай актуальный `.zip` из [Releases](../../releases)
2. Помести в папку `resourcepacks` внутри `.minecraft`
3. Включи в игре: **Настройки → Ресурспаки**

Пак поддерживает [GTNH Resource Pack Updater](https://github.com/GTNewHorizons/GTNHLib/pull/260) — при наличии обновления игра уведомит тебя прямо в игре.

## Участие в переводе

PR приветствуются. Ключи локализации находятся в `assets/<аддон>/lang/ru_RU.lang`.
